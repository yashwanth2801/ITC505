Palindrome Checker

Enter a word or phrase, and we will check if it is a palindrome:

Addendum: This page checks if a word or phrase is a palindrome by cleaning the input and reversing it. The application requests data input from a single text input box. The program first removes all non-alphanumeric characters and converts every input to lowercase before the text reversal process. The processing system returns the modified text which the program uses to compare against the original cleaned input. The input qualifies as a palindrome when the two sequences match. In all other cases the input does not represent a palindrome. The program shows the output dynamically on the webpage.
