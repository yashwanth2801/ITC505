Addendum

TThe purpose of his lab was to develop a mad lib form through ExpressJS. The form contained five input fields which I submitted using a POST request.

I used npm to install Express and Morgan onto my system. I established a public folder for serving HTML and CSS files as part of my project setup.

The server.js file processed both GET and POST request operations. The server generates a complete mad lib that contains user-submitted values when the form submission is processed.

I loaded the page through http://localhost:8080/ITC505/lab-7/index.html and all functions operated as intended.

The server funcionality was checked through the route do_a_random which generated random numbers.
