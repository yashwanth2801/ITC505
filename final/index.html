Lights Out

Addendum

Process & Creative Choices:
I built a 5x5 dynamic grid system through JavaScript code before adding embedded CSS style elements and added a button toggle function for each grid cell. A single click operation changes each cell from black to white and vice versa. The “off” (black) state received representation through a class named “is-off”. The winning condition is satisfied when all cells turn white.

Solvable Randomization:
The board remained solvable after using white cells along with randomized cell activations. Every simulation mirrors authentic gameplay by keeping its solvable state but creating different arrangements for each game.

Challenges & Solutions:
The main difficulty emerged when trying to randomize the board while also avoiding unsolvable puzzle conditions. User-generated board clicks served as my solution to addressing this problem instead of using random state changes. The application required extra attention regarding debugging element behaviors with state preservation throughout the board through dataset attributes.

Insights Gained:
Working on this project allowed me to deepen my understanding of how to manipulate the DOM along with class toggling methods and event handling in addition to UI logic development. My studies taught me to maintain game logic systems which follow user expectations and interface requirements.
